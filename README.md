# `react-router-dom` + `github-pages`

## Install

```
yarn add react-router-dom
yarn add --dev gh-pages
```

## Build

```
yarn react-scripts build
```

## Deploy

```
yarn gh-pages -d build
```

## Routing

```
{
  path: "/",
  View: Page1,
},
{
  path: "/page2",
  View: Page2,
},
{
  path: "/page3",
  View: Page3,
},
```
